## Installation

The Float Subgraph Uncrashable can be installed with `npm` or `yarn`:

```sh
# NPM
npm install @float-capital/float-subgraph-uncrashable

# Yarn
 yarn add @float-capital/float-subgraph-uncrashable
---
```
